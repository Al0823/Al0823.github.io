JavaScript Dates Tutorial

This page teaches you how to work with Date objects in JavaScript. You can create a Date object to get the current date and time, or set a specific date.





Date objects are static. The "clock" is not "running".

The computer clock is ticking, date objects are not.


JavaScript Date Output


By default, JavaScript will use the browser's time zone and display a date as a full text string:

Fri Feb 06 2026 06:29:43 GMT-0500 (Eastern Standard Time)

You will learn much more about how to display dates, later in this tutorial.

Examples

Creating a Date Object

Use the Date() constructor:

let today = new Date();
console.log(today);

Getting the Current Date and Time

You can use methods like getFullYear(), getMonth(), getDate(), getHours(), etc.

let now = new Date();
let year = now.getFullYear();
let month = now.getMonth() + 1; 
let day = now.getDate();
let hour = now.getHours();
let minute = now.getMinutes();

console.log(`Today is ${month}/${day}/${year} ${hour}:${minute}`);

Setting a Specific Date

You can create a Date object for a specific date:

let birthday = new Date("2005-06-15");
console.log(birthday);

Other Useful Date Methods

  • getDay() – returns the day of the week (0–6)
  • getTime() – returns milliseconds since January 1, 1970
  • setFullYear(year), setMonth(month), setDate(day) – modify a date
