"Assignment" Javascript tutorial



This is the "Assignment" Javascript page, it teaches you how to write this code and how it works. Now what this code can do is change what some text says, it can also make buttons and just possibly redirects you if you know what you are doing.

Examples



The simple assignment operator (=) assigns a simple value to a variable.

The addition assignment operator adds a value to a variable.

The subtraction assignment operator subtracts a value from a variable.

The multiplication assignment operator multiplies a variable by whatever is where 5 is located.

The exponentiation assignment operator raises a variable by in this case what would be "5"

The Division Assignment Operator divides a variable by 5 in this case.

The remainder operator assigns a remainder to a variable.

The basic assignment operator assigns values to variables.

The addition assignment operator can be used to add strings.

LOGICAL AND ASSIGNMENT

The Logical and Assignment operator is used between two values.
If the first value is correct/true, then the second value is assigned.

LOGICAL OR ASSIGNMENT

The Logical and Assignment operator is used between two values.
If the first value is incorrect/false, then the second value is assigned.

The Nullish Coalescing Assignment operator is used between two values. If the first value is not defined or null, the second value is assigned.

The "..." operator splits iterables into individual elements.
