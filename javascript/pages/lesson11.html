Arithmetic Javascript tutorial



This is the "Arithmetic" Javascript page, it teaches you how to write this code and how it works. Now what this page does is go over different Arithmetic Operators, some of these operators are "+" "-" "*" "**" "/" "%" "++" "--". These Arithmetic operators are Addition, Subtraction, Multiplication, Exponentiation, Division, Modulus/Remainder, Increment and Decrement.

Examples



The Average Arithmetic operation takes two numbers and makes a new one.

Arithmetic operations can also be used done with numbers being represented as variables.

You can also use expressions to produce a new number.

The add operator adds numbers.

The subtraction operator subtracts numbers

The multiplication operator multiplies numbers

The division operator divides numbers

The modulus operator returns the division remainder.

The Increment operator increments the numbers

The Decrement Operator decreases numbers.

The exponentiation operator raises the first operand to the power of the second operand.

"Math.pow(x,2)" makes the same result as "x**y"
