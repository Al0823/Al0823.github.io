This is the "Data Types" Javascript page, it teaches you about this code and how it works.

Different kinds of Data Types



There are "String," "Number," "Bigint," "Boolean," "Object," "Undefined," "Null," and "Symbol."

Examples



Data Types Examples

{ // Strings let color = "Yellow"; let lastName = "Johnson"; // Number let length = 16; let weight = 7.5; // BigInt let x = 1234567890123456789012345n; let y = BigInt(1234567890123456789012345) // Boolean let x = true; let y = false; // Object const person = {firstName:"John", lastName:"Doe"}; // Array object const cars = ["Saab", "Volvo", "BMW"]; // Date object const date = new Date("2022-03-25"); // Undefined let x; let y; // Null let x = null; let y = null; // Symbol const x = Symbol(); const y = Symbol(); }

When adding a number and a String, Javascript will treat the number as a string.

JavaScript evaluates expressions from left to right. Different sequences can produce different results as seen above.
In the first example, JavaScript treats 6 and 7 as numbers, until it reaches "Pontiac".
In the second example, since the first thing is a string(Pontiac), all things after are treated as strings.

JavaScript has dynamic data types. This means that the same variable can be used to hold different data types.

A string (or a text string) is a series of characters like "Pontiac GTO." Strings are written with quotes, single or double quotes work for this.

You can use quotes inside a string, as long as they do not match the quotes around the String.

Numbers can be written in any way possible, be it with or without decimals.

Extra big or extra small numbers can be written with "scientific" notation.

You cannot perform math between a BigInt type and a Number type.

You can not perform math between a BigInt data type and a Number data type.

Javascript based booleans can have 2 values, true or false.
