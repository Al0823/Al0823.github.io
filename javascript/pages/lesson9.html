CONST Javascript tutorial



This is the "CONST" Javascript page, it teaches you how to write this code and how it works.
Variables declared with "CONST" are very specific; they can not be in a block scope, be reassigned or redeclared.
When should I use "CONST?"
You should use "CONST" when you declare a new Array, Object, Function, or RegExp.

Examples




"CONST" variables can not be reassigned, they must also be assigned a value before being declared.
