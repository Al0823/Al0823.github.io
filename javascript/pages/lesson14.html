Parameters vs. Arguments


Parameters and Arguments are distinct ideas when working with FUNCTIONS.
Parameters are the names listed in the FUNCTION definition.
Parameters are the names of the values that will be passed.
Arguments are values that get passed to the function when it is called.
Arguments are the values returned by the FUNCTION.

"name" and "age" are parameters
"Aloysius" and 17 are Arguments
