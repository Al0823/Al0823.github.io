JavaScript Booleans Tutorial

A Boolean represents one of two values: true or false. Booleans are mainly used for decision-making in code.

Boolean Values

let isCodingFun = true;
let isMathBoring = false;

console.log(isCodingFun);
console.log(isMathBoring);
