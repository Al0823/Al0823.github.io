JavaScript Random Numbers Tutorial

JavaScript uses the Math.random() method to generate random numbers. This is commonly used in games, simulations, and decision-making logic.

Basic Random Number

Math.random() returns a number between 0 (inclusive) and 1 (exclusive).

let num = Math.random();
console.log(num);

Random Whole Numbers

To get a random whole number, combine Math.random() with Math.floor().

// Random number from 0 to 9
let num = Math.floor(Math.random() * 10);

Random Number in a Range

// Random number from 1 to 100
let num = Math.floor(Math.random() * 100) + 1;
