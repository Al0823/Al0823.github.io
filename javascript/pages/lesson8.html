LET Javascript tutorial



This is the "LET" Javascript page, it teaches you how to write this code and how it works. Now what this code can do is change what some text says, it can also make buttons and just possibly redirects you if you know what you are doing.

EXAMPLES




"let" statements will not work outside of the squiggly brackets. If it was working, x would equal 2. They do not work because they are apart of the block scope.


"let" statements will not work outside of the squiggly brackets but "var" statments will. This is because "VAR" statements are old, also because they are apart of the global scope.

Variables defined with "let" cannot be redeclared.

Variables defined with "var" can be redeclared. Redeclaring a variable with "var" inside a block will also redeclare the variable out side the block, this can impose problems. But using a "let" statement instead can fix this issue.
