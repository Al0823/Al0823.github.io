JavaScript Date Get Methods Tutorial

The get methods are used to extract parts of a Date object, such as the year, month, day, hour, and more.

Common Date Get Methods

  • getFullYear() – returns the year (e.g., 2026)
  • getMonth() – returns the month (0–11)
  • getDate() – returns the day of the month (1–31)
  • getDay() – returns the day of the week (0–6)
  • getHours() – returns the hour (0–23)
  • getMinutes() – returns minutes (0–59)
  • getSeconds() – returns seconds (0–59)
  • getMilliseconds() – returns milliseconds (0–999)
  • getTime() – returns milliseconds since Jan 1, 1970
