JavaScript Date Set Methods Tutorial

The set methods allow you to change parts of a Date object, such as the year, month, day, or time.

Common Date Set Methods

  • setFullYear(year) – sets the year
  • setMonth(month) – sets the month (0–11)
  • setDate(day) – sets the day of the month
  • setHours(hour) – sets the hour
  • setMinutes(min) – sets the minutes
  • setSeconds(sec) – sets the seconds
  • setMilliseconds(ms) – sets milliseconds
  • setTime(ms) – sets date using milliseconds since Jan 1, 1970
