JavaScript Math Object Tutorial

The Math object allows you to perform mathematical tasks in JavaScript. It has properties (like PI) and methods (like round, random, and square root).

Math Properties

  • Math.PI – returns 3.141592653589793
  • Math.E – returns Euler's number
  • Math.SQRT2 – square root of 2

Common Math Methods

  • Math.round(x) – rounds to nearest integer
  • Math.ceil(x) – rounds up
  • Math.floor(x) – rounds down
  • Math.trunc(x) – removes decimal part
  • Math.pow(x, y) – x to the power of y
  • Math.sqrt(x) – square root
  • Math.abs(x) – absolute value
  • Math.min(...numbers) – smallest number
  • Math.max(...numbers) – largest number
  • Math.random() – random number between 0 and 1
