String Search Javascript tutorial



This is the "String Search" Javascript page, it teaches you how to write this code and how it works.

Now what a "String Search" is in Javascript is it is the process of finding a certain substring or pattern inside a larger string.



Examples



JavaScript String indexOf()


The indexOf() method returns the index (position) of the first occurrence of a string in a string, or it returns -1 if the string is not found.




Javascript counts positions from zero

0 is the first position in a string, 1 would be the second and so on so forth.


JavaScript String lastIndexOf()


The lastIndexOf() method returns the index of the last occurrence of a specified text in a string




Both indexOf(), and lastIndexOf() will return -1 if the text is not found.



Both of these methods accept a second parameter as the starting position for the search.



The lastIndexOf() methods searches backwards from end to start. Which means if the second parameter is 300, the search starts at position 300, and searches to the Start of the string.

Javascript String Search()


The search() method searches a string for a string or a basic expression and returns the position of the match.






You may have noticed that indexOf() and search() are almost identical in how they look.

They are NOT the same, they have differences; these differences are:

The search() method cannot take a second start position argument.

The indexOf() method cannot take powerful search values also known as regular expressions.

Javascript String match()


The match() method returns an array containing the results of matching a string against a string also known as a regular expression










If a regular expression lacks the g (global) modifier, the match() method returns only the first match found in the string.

String matchAll( )




When using a normal expression as the parameter, the global flag (g) is needed; otherwise, a TypeError will be thrown.




To perform a case-insensitive search, the insensitive flag (i) must be included.



The matchAll( ) method was introduced in 2020. It is not supported in Internet Explorer.


String includes( )
The includes( ) method returns true if a string contains the specified value; otherwise, it returns false.



The includes( ) method is case-sensitive. It was introduced in 2015

String startsWith( )
The startsWith( ) method returns true if a string starts with a specified value; otherwise, it returns false.





You can also specify a starting position for the search.






The startsWith( ) method is case-sensitive. It was introduced in 2015.


String [URLendsWith( )
The endsWith( ) method returns true if a string ends with the specified value; otherwise, it returns false.
